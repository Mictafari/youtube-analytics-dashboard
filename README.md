# youtube-analytics-dashboard
Excel dashboard analyzing my YouTube channel performance using Power Query, Pivot Tables, and KPIs

 📺 YouTube Analytics Dashboard (2023–2026) - For Kbroda Tech YouTube Channel

 📌 Project Overview

This project is an **interactive Excel dashboard** built from my personal YouTube channel data (Kbroda Tech). I used **Power Query** to clean and transform raw analytics exports, **Pivot Tables** to aggregate metrics, and **Pivot Charts** with **slicers** to visualize performance trends across 2023–2026.

The dashboard answers key questions like:
- Which months had the highest views and revenue?
- What are my top 10 performing videos?
- How does CTR and impressions trend over time?
- What is my subscriber growth trajectory?

🎯 Problem Statement

YouTube Studio provides raw CSV exports, but they are scattered across multiple files and lack a unified view. Manually tracking views, revenue, watch time, and CTR across years is time-consuming and error-prone. This dashboard solves that by consolidating all data into a single, interactive report that updates with new exports.

🛠️ Tools & Skills Demonstrated

- Power Query** - Cleaned, transformed, and appended 3+ years of YouTube data
- Pivot Tables** - Aggregated views, revenue, watch time by month, year, and video
- Pivot Charts** - Visualized monthly trends, top videos, and CTR analysis
- Slicers** - Added interactivity to filter by date range
- KPI Cards** - Displayed key metrics (total views, revenue, subs, watch time)
- Excel Formatting** - Professional dashboard layout, conditional formatting, color coding 

📊 Dashboard Features

- Yearly Summary** - Metrics broken down by year (2023–2026)
- Monthly Trend** - Line chart showing views and watch time month-over-month
- Top 10 Videos** - Bar chart of highest-performing videos by views
- Impressions & CTR** - Dual-axis analysis of reach and click-through rate
- Slicers** - Interactive filters for year and month selection
